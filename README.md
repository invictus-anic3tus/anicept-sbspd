## About

Hi! I'm Anicetus, and I love 3D printers and making custom hardware. Some of my favorite hardware to make is custom PCBs, namely the [Anicept Tempus](https://github.com/invictus-anic3tus/tempus), which often use very tiny SMD parts. The common choice is to go buy a solder stencil, which allows solder paste to be spread onto the PCB and easily reflowed. But stencils can get _expensive_. And, of course, they can only be used for one specific PCB! If your project needs a redesign, that means you need a whole new stencil for it.

So I got to thinking. Way back when, all plastic parts were made with injection molding, which had the same problems as solder stencils do now: to make a different plastic part, you need a whole new mold, which is very expensive. Then they invented 3D printers. These devices move in all three dimensions to make almost _any_ plastic part you want--why shouldn't the stencil problem have the same solution?

Well, maybe there is a good reason that this isn't so widespread. In any case, I'm about to find out.

## Features

- A Nema 11 stepper motor for accurate and high-power control over the solder syringe
- An easy-to-use clamp/hinge system that holds everything tightly in place
- A magnet-based mounting system that's compatable with any 3D printer
- Only four different screws/nuts!
- A fully 3D printable case
- A compact form factor
- ... all this for around $20!
